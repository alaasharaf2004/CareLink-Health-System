2) إصلاحات وتجربة مستخدم عامة

2.1 مشكلة التمرير التلقائي للدردشة إصلاح

  • السبب: صندوق الدردشة كان يستدعي scrollIntoView على الصفحة كاملة.
  • الحل: التمرير داخل قائمة الرسائل فقط، وتثبيت الصفحة أعلى عند فتح تفاصيل الموعد.
  • الملفات: AppointmentChatBox.jsx، صفحات تفاصيل الموعد للطبيب والمريض.

3) أنيميشن وتفاعل اللوحات

تم توحيد طبقة حركة للطبيب / المريض / الصيدلي:

  • ظهور تدريجي لعناصر القائمة الجانبية
  • تفاعل الهيدر وبطاقة الملف الشخصي
  • بطاقات سريعة بلمعان ورفع عند المرور
  • انتقال صفحات عبر DoctorPageShell / PatientPageShell / StaffPageShell
  • هيرو ترحيبي للصيدلية PharmacyWelcomeHero

ملف الأنماط: src/index.css (كلاسات workspace-*)

4) صفحة تفاصيل موعد الطبيب

العنصرالتعديل
ترتيب الصفحة بيانات المريض → ملخص → تشخيص/تحاليل → وصفة + دردشة جنب بعض → إنهاء الزيارة بالأسفل
الدردشة بجانب الوصفة الطبية (عمودين) بدل أسفل الصفحة فقط
إنهاء الزيارة زر فعّال يحدّث الحالة لـ مكتمل + إشعار للمريض + أنيميشن احتفالي
طلب أشعة قسم جديد «إرسال للأشعة» مع حالة awaiting_radiology

5) دور فني الأشعة (Radiology)

دور جديد تمت إضافته في كل الأماكن اللازمة:

  • تسجيل الدخول (طاقم العيادة) + توجيه إلى /radiology/dashboard
  • لوحة أشعة: استقبال الطلبات، إدخال تقرير، PDF اختياري، إشعار المريض
  • إدارة الكوادر (تبويب الأشعة)
  • نظام البث (استهداف الأشعة)
  • المتجر المحلي: imagingOrders + حساب تجريبي
  • عرض الأشعة في صفحة المريض

حساب تجريبي: radiology@carelink.com / 123456

6) تسجيل الدخول لطاقم العيادة

  • إضافة خيار «فني أشعة» ضمن أدوار الطاقم
  • تصميم اختيار الأدوار: شبكة 2 فوق و 2 تحت بحجم مضغوط (شكل التبويبات السابقة)

7) ميزات موظف الاستقبال (7 إضافات)

#الميزةالوصفأين؟
1 شاشة غرفة الانتظار عرض رقم الدور والمريض الحالي والتالي على شاشة TV /reception/waiting-display
2 مرضى تابعون ربط طفل/تابع بولي أمر عند التسجيل أو من ملف المرضى تسجيل مريض + ملف المرضى
3 تسليم/تسلّم الوردية ملاحظات تنتقل بين الورديات مع تأكيد الاستلام لوحة اليوم (جانب الطابور)
4 أعلام تنبيه حساس / يحتاج وقتاً أطول / غالباً يتأخر + ملاحظة الطابور + ملف المرضى
5 حالة التأمين نشط / منتهٍ / بدون / غير محدد + شركة التأمين التسجيل + تنبيه/تأمين
6 ملخص اليوم إجمالي، مكتمل، ملغى، لم يحضر، حضور، عند الطبيب… زر «ملخص اليوم»
7 اختصارات لوحة مفاتيح Ctrl+N تسجيل · Ctrl+Shift+S ملخص · Ctrl+Shift+W شاشة انتظار لوحة الاستقبال

كذلك: زر إنهاء زيارة للاستقبال مع إشعار للمريض.

8) شاشة غرفة الانتظار — إعادة التصميم

  • خلفية فاتحة بهوية CareLink (بدلاً من الوضع الداكن)
  • شعار CareLink + ساعة وتاريخ
  • مساحة كبيرة: «يُدعى الآن» مع رقم الدور واسم المريض والطبيب
  • قائمة «التالي» على الجانب
  • حركة خفيفة عند تغيّر المريض الحالي
